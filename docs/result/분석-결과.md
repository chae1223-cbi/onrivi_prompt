## 2. 데이터 흐름도 (Data Flow)

| 단계 | 발신처 (Source) | 수신처 (Target) | 매개 데이터 (Data) | 처리 내용 |
| :--- | :--- | :--- | :--- | :--- |

## 3. 엣지 케이스 및 예외 처리 (Edge Cases)

- **E1. 잘못된 데이터 형식:** API 요청 시 JSON 형식이 올바르지 않거나 필수 필드(email 등) 누락 시 400 Bad Request 처리.
- **E2. 인증되지 않은 접근:** (현재 결함) 관리자 API 세션/인증 부재. 향후 토큰 기반 인증 필터 추가 필요.
- **E3. 중복 키 충돌:** 라이선스 키 생성 시 DB Unique 제약 조건 위반 발생 시 재시도 로직 필요.
- **E4. 런타임 환경 제약:** Cloudflare Pages Functions의 엣지 환경에서 지원하지 않는 Node.js 라이브러리 호출 시 고립화 방지.
