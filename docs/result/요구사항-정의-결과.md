## 1. 핵심 가치 도출 (Core Value)

**"1인 지식 창업자의 프롬프트 자산을 상용화할 수 있는 인프라 제공"**

- **문제 정의:** 1인 기업가나 크리에이터가 공들여 만든 프롬프트(Prompt)를 단순 공유가 아닌 '상품'으로서 안전하게 판매하고 라이선스를 관리할 수 있는 도구가 부재함.
- **해결 방안:** 복잡한 백엔드 구축 없이도 라이선스 키를 발급하고 검증할 수 있는 서버리스 인프라와 프리미엄 브랜드 이미지를 구축할 수 있는 랜딩 페이지 패키지 제공.

## 2. MVP 형태 제안 (MVP Strategy)

### **[추천] 싱글 코어 기능 MVP (Single Core Function MVP)**
단순 랜딩 페이지를 넘어, 실제 라이선스 비즈니스가 돌아가는 '최소한의 운영 도구'를 완벽하게 구현하는 형태를 지향합니다.

- **필수 포함 기능:**
  - **라이선스 제어부:** 관리자 페이지를 통한 키 생성 및 이력 관리 (`src/pages/Admin.tsx`)
  - **검증 로직 API:** 클라이언트 앱에서 키 유효성을 확인할 수 있는 서버리스 엔드포인트 (`functions/api/verify.ts`)
  - **브랜드 쇼케이스:** 잠재 고객에게 신뢰를 줄 수 있는 프리미엄 랜딩 페이지 (`src/App.tsx`)

## 3. 측정 지표 설정 (Key Metrics)

가설 검증을 위해 '허상 지표'가 아닌 실제 비즈니스 임팩트를 측정합니다.

1. **라이선스 발행 및 활성화 수 (LTV 기반):** 수동/자동으로 발행된 라이선스가 실제 고객에 의해 얼마나 활성화되었는지 측정.
2. **랜딩 페이지 전환율 (CTR):** 메인 Hero 섹션에서 'Pricing' 또는 'Demo'로 넘어가는 방문자의 비율 측정.
3. **API 호출 성공률:** 엣지 런타임에서의 안정적인 라이선스 검증 여부 모니터링.

---
**보고:** 현재 OnriviPrompt는 기술적으로 TanStack Query 기반의 안정적인 관리자 UI와 Cloudflare D1 기반의 서버리스 라이선스 API를 갖추고 있어, 실제 기획 가설을 즉시 검증할 수 있는 수준의 MVP 상태입니다.
