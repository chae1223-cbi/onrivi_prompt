# 뉴뉴스메일(NewNewsMail) 개발 계획서 v1.0

## 1. 개요
본 프로젝트는 인공지능 에이전트(Anti-gravity)를 핵심 동력으로 활용하여 뉴스 수집, 요약, 이미지 합성 및 이메일 발송의 전 과정을 자동화하는 지능형 뉴스레터 서비스 구축을 목표로 합니다. 사용자는 복잡한 운영 업무에서 벗어나 서비스 설계 및 비즈니스 전략에 집중할 수 있는 '에이전틱 워크플로우'를 지향합니다.

### 1.1 핵심 가치
- **에이전틱 자동화**: 사람이 개입하지 않는 자율 구동형 파이프라인 구축
- **법적 리스크 제로**: 개인정보 보호법 및 광고성 정보 전송 규정 엄격 준수
- **지능형 요약**: 사용자 맞춤형 3단계 요약 및 결론 우선 정보 제공
- **비주얼 경험**: AI 기반 이미지 생성 및 한글 타이포그래피 자동 합성

## 2. 시스템 아키텍처
본 시스템은 에이전트-그리드 구조를 기반으로 하며, 각 모듈이 자율적으로 통신합니다.

### 2.1 에이전트 엔진
1. **수집 에이전트(Intake)**: 중복 제거 및 카테고리 분류 자동화
2. **요약 에이전트(Summary)**: Gemini 1.5 Flash 기반 3단계 요약 생성
3. **비주얼 에이전트(Visual)**: Pollinations.ai 및 Pillow를 활용한 콘텐츠 커버 생성
4. **발송 에이전트(Dispatch)**: Brevo API를 통한 개인화 뉴스레터 전송

### 2.2 기술 스택
- **언어**: Python, TypeScript
- **프레임워크**: FastAPI (Backend), Next.js (Frontend)
- **데이터베이스**: SQLite (newnews.db)
- **AI/LLM**: Google Gemini 1.5 Flash, Pollinations.ai
- **외부 연동**: Brevo (Email), Toss Payments (Billing)
