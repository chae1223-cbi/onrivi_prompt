| 프로그램명 | 역할 및 기능 설명 | 사양서 링크 |
| :--- | :--- | :--- |
| **`scheduler_agent.py`** | **전체 오케스트레이터**. DB 스케줄에 따라 각 에이전트 실행 관리 | [사양서](file:///d:/Antigravity/NewsLetters/docs/specs/스케줄러_에이전트_명세서.md) |
| **`base_agent.py`** | **공통 기반 클래스**. DB 연결, 로깅, 설정 로드 등 공통 기능 제공 | [사양서](file:///d:/Antigravity/NewsLetters/docs/specs/기본_에이전트_명세서.md) |
| **`intake_agent.py`** | 뉴스 수집 에이전트. RSS 및 구글 뉴스를 통해 원문 확보 | [사양서](file:///d:/Antigravity/NewsLetters/docs/specs/데이터_수집_에이전트_명세서.md) |
| **`summary_agent.py`** | AI 요약 에이전트. 도메인별 3단계 요약 및 인사이트 생성 | [사양서](file:///d:/Antigravity/NewsLetters/docs/specs/요약_에이전트_명세서.md) |
| **`visual_agent.py`** | 이미지 에이전트. Pollinations AI를 이용한 콘텐츠 커버 생성 | [사양서](file:///d:/Antigravity/NewsLetters/docs/specs/시각화_에이전트_명세서.md) |
| **`dispatch_agent.py`** | 발송 에이전트. 개인화된 레터를 구독자에게 최종 전송 | [사양서](file:///d:/Antigravity/NewsLetters/docs/specs/발송_에이전트_명세서.md) |

## 5. 운영 및 데이터 (Ops & Data)
- **`main_agent.py`**: 시스템을 일괄 가동하는 엔트리 포인트 스크립트.
- **`scripts/utils.py`**: KST 시각 처리, DB 상태 업데이트 등 공통 함수.
- **`db/NewNewsMail.db`**: 모든 기사, 사용자, 로그가 저장되는 SQLite 데이터베이스.
- **`.env`**: GEMINI_API_KEY, BREVO_API_KEY 등 핵심 보안 설정.
