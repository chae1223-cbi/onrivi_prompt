# 뉴뉴스메일(NewNewsMail) 기술 스택 정의서

대표님, 서비스의 안정적인 운영과 향후 유지보수를 위해 선정된 **표준 기술 스택 및 라이브러리 목록**입니다. 모든 도구는 1인 기업의 '가성비'와 '성능'을 최우선으로 고려했습니다.

---

## **📂 [뉴뉴스메일] 표준 기술 스택**

### **1. Core Language & Framework**
*   **Python 3.10+**: 에이전틱 워크플로우 및 데이터 처리 핵심 언어.
*   **FastAPI**: 고성능 비동기 API 서버 (구독 웹훅 및 대시보드 백엔드).
*   **Uvicorn**: ASGI 서버 서버 환경 구축.

### **2. AI & External APIs**
*   **Google Gemini 1.5 Flash**: 뉴스 분석 및 3단계 요약 (가장 빠른 속도와 가성비).
*   **Pollinations.ai**: AI 이미지 생성 (API 키 기반 고품질 배경 추출).
*   **스티비 (Stibee) API** -> **Brevo API**: 글로벌 및 국내 발송 최적화 (일 300통 무료, Python SDK 지원).
*   **Toss Payments SDK**: 정기 결제(빌링) 및 웹훅 연동.

### **3. Data & Storage**
*   **SQLite 3**: 파일 기반의 가벼운 로컬 DB (1인 기업 프로젝트에 최적).
*   **SQLAlchemy / Tortoise-ORM**: DB 모델링 및 쿼리 관리 자동화.

### **4. Visualization & Utilities**
*   **Pillow (PIL)**: 이미지 내 한글 텍스트 레이어 합성 및 리사이징.
*   **BeautifulSoup4 / Playwright**: 뉴스 사이트 데이터 크롤링 및 텍스트 추출.
*   **Pydantic**: API 요청/응답 데이터 검증 및 타입 안전성 확보.

---

### **📦 주요 라이브러리 설치 명령 (pip)**
```bash
pip install fastapi uvicorn google-generativeai pillow requests beautifulsoup4 pydantic sqlalchemy toss-payments
```

---

### **💡 뉴뉴스메일 AI를 위한 가이드**
에이전트는 코딩 시 위 스택을 최우선으로 사용해야 합니다. 특히 이미지 합성 시에는 `Pillow`를, 데이터 검증 시에는 `Pydantic`을 활용하여 코드 정합성을 유지합니다.
